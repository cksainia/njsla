# NJSLA-Adaptive Grade 3 Practice

A single-file Progressive Web App that mocks the 2026 **NJSLA-Adaptive** Grade 3 assessments
(Mathematics, ELA-Reading, ELA-Writing) with computer-adaptive difficulty, NJSLA-style 650–850
scale scoring, save/resume, on-device analytics, and optional live question generation via your
choice of **Anthropic (Claude)**, **OpenAI (ChatGPT)**, or **Google Gemini** (key stored only on
the device). Works fully offline using a built-in 221-question bank.

Live: https://cksainia.github.io/njsla/

Files: `index.html` (all app logic inline), `data.js` (question bank), `sw.js`, `manifest.webmanifest`, `icon.svg`.
