# NJSLA-Adaptive Grade 3 Practice Test
A faithful, Cambium-style Grade 3 NJSLA-Adaptive practice web app: pre-test flow, test-player chrome
(item map, two-pane passages, per-item tools), multiple item types (MC, multi-select, numeric/fraction
entry with on-screen keypad, multi-part, bar-graph builder, extended writing), a documented group/MST
adaptive engine, and a parent report. Single-file build; `index.html` + `data.js`. Live: https://cksainia.github.io/njsla/
